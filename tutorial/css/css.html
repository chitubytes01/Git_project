Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML elements. It allows you to control layouts, colors, fonts, spacing, and animations.

Setup

You can write CSS in three ways: Inline, Internal, and External. For detailed guide: Open PDF Download PDF

Selectors


/* Select by element */
p {
  color: blue;
}

/* Select by class */
.container {
  padding: 20px;
}

/* Select by id */
#main {
  margin: 10px;
}

Colors


body {
  background-color: #020617;
  color: #e5e7eb;
}

Fonts


p {
  font-family: 'Segoe UI', sans-serif;
  font-size: 16px;
}

Box Model


div {
  width: 200px;
  padding: 10px;
  border: 2px solid #38bdf8;
  margin: 10px;
}

Positioning


.header {
  position: fixed;
  top: 0;
  width: 100%;
}

Flexbox


.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Grid


.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Animations


@keyframes slideUp {
  from { opacity: 0; transform: translateY(40px); }
  to { opacity: 1; transform: translateY(0); }
}

.element {
  animation: slideUp 1s ease;
}

Responsive Design


@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
